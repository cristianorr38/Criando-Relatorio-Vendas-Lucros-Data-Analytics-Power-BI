## 📌 Sobre o Projeto
Este projeto faz parte do **Desafio de Projeto - Criando um Relatório de Vendas e Lucros com Data Analytics com Power BI**.  
O objetivo foi **modificar e aprimorar relatórios financeiros e de vendas**, aplicando conceitos de design e usabilidade, como:
- Posicionamento
- Contraste
- Proporção áurea
- Segmentação dos dados
- Criatividade na quebra de regras visuais

O resultado final é um relatório interativo e estilizado, com navegação intuitiva e foco na experiência do usuário.
